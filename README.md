# Prodip Das - Professional Portfolio

> International Event Management Professional & English Language Specialist

## Overview

This is the personal portfolio website for **Prodip Das**, showcasing professional experience, skills, education, and portfolio in an elegant, modern design with dark/light theme support.

## Features

### 🎨 Advanced Features
- **Dark/Light Theme Toggle** - Switch between themes with persistent preference saved in localStorage
- **Responsive Design** - Mobile-first approach with breakpoints for all screen sizes
- **Scroll Animations** - Smooth reveal animations using ScrollReveal.js
- **Custom Cursor Effect** - Ambient glow effect that follows mouse movement
- **Progress Indicator** - Top progress bar showing scroll position
- **Preloader** - Elegant loading spinner for better UX

### ✨ Portfolio Sections
- **Hero** - Eye-catching introduction with profile image and social links
- **About** - Executive summary with statistics and career objective
- **Experience** - Professional journey with detailed role descriptions
- **Portfolio** - Project showcase with images and descriptions
- **Testimonials** - Client reviews and recommendations
- **Education** - Academic timeline with degree details
- **Contact** - Direct contact information and social media links

### 🛠️ Technical Features
- **SEO Optimized** - Comprehensive meta tags, Open Graph, Twitter Cards
- **Structured Data** - JSON-LD schema for better search visibility
- **Accessibility** - Skip links, ARIA attributes, keyboard navigation
- **Performance** - Preconnect, preload, optimized rendering
- **Cross-Browser** - Modern CSS with fallbacks
- **Print Support** - Professional print stylesheets

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid, Animations
- **JavaScript (ES6+)** - Modern vanilla JS
- **Font Awesome 6** - Icon library
- **ScrollReveal** - Animation library

## Getting Started

### Prerequisites
- A modern web browser
- No build tools required - pure static website

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser

```bash
# If you have git
git clone https://github.com/ShovonScripts/PRODIPDAS.git

# Navigate to the directory
cd PRODIPDAS

# Open in browser
open index.html  # macOS
start index.html  # Windows
```

### Deployment

This website can be deployed to any static hosting service:

- **GitHub Pages** - Free hosting for static sites
- **Netlify** - Drag and drop deployment
- **Vercel** - Instant deployment
- **Cloudflare Pages** - Global CDN

## Customization

### Update Profile Image
Replace `prodip.jpg` with your own profile image.

### Update Content
Edit `index.html` to update:
- Personal information
- Work experience
- Education details
- Skills and competencies
- Portfolio projects
- Testimonials

### Change Color Theme
Modify the CSS variables in `style.css`:

```css
:root {
    --primary: #6366f1;
    --primary-dark: #4f46e5;
    --accent: #c084fc;
    /* ... */
}
```
